# Clean Sweep – by Business Data Gurus

Clean Sweep is an enterprise-ready system maintenance and onboarding tool for Windows. Designed to optimize system performance, clean unnecessary files, and prepare machines for new hires—all with zero tech experience required.

## 🔧 Features
- Auto-scans laptops on launch and login
- Identifies large files, old apps, and viruses
- Optional app uninstaller and optimization engine
- Preps laptops for new hires with clean system state
- Works across VPN on a company network
